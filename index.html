Intro

This webpage is part of a school assignment. The goal is to use HTML5 properly with semantic tags, make it accessible, and follow basic SEO rules. No CSS or JavaScript is added, just clean HTML.

Accessibility

Accessibility means building websites that work for everyone, including people using screen readers or with disabilities. Using the right tags makes it easier for those tools to read and understand the page.

  • Using clear tags like header, aside and main
  • Headings should follow the right order (H1, H2, etc.)
  • Links should make sense even if read out loud
